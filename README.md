# الشاهين كاش - مشروع كامل

## تشغيل Frontend
cd frontend
npm install
npm run dev

## تشغيل Backend
cd backend
npm install
npm start

## ملاحظات
- ربط قاعدة بيانات MongoDB عبر ملف .env
- تصميم متوافق مع الموبايل وDesktop
- دعم متعدد اللغات
- ملفات الصور موجودة في frontend/public/assets
- يمكن تحويل Frontend إلى APK باستخدام Capacitor أو WebView
